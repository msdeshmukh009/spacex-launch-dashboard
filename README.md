## **About the app -**

- Authentication using auth0.
- All the launches are listed.
- Filter on basis of Past and upcoming.
- Filter on basis of start and end date.
- Check the launch info on modal.

## **Tech Stack -**

- React JS
- Redux-toolkit
- Auth0
- Tailwind CSS
